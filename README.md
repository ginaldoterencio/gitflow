# GIT FLOW Snippets
Pacote de snippets para processos do gitflow

**Instalação**

```bash
npm install -g gitflow-snippets
```

**Cria novo branch**

```bash
gitflow create-branch [nome-do-branch]
```

**Faz merge no branch com o working**

```bash
gitflow deploy [nome-do-branch]
```
